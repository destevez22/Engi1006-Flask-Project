Homework 1

The written portion served as an introduction to basic programming principles, as well
as motivation for creating an algorithm based mindset. The coding portion revealed just how
important this is, as all of our work in python can be described as a bunch of small tasks
written as code that come together to make a program. The us of lists in the second portion
was incredibly useful for exploring syntax and the power of python lists, something we used a lot.

Homework 2

We continued to develop our understanding of basic coding theory with Homework 2, especially
when it comes to time complexity, which is at first a rather weird concept. It was also tied
into the different types of sorting algorithms there are, which can be quite useful. The coding
of this assignment introduced us to simulations and graphing with python, which have evident uses
in many scenarios. We also began to set up our machine learning algorithm and thus learned about
dictionaries and reading csv files.

The first graph shows the trajectory for a projectile in motion. The second graph reveals that the simulated value and expected value do, in fact, diverge, if only slightly in this particular case.

Homework 3

The theory part of this assignment served to further solidify the concepts of time complexity,
as well as introduce the topic of recursion. Recursion is an elegant concept, which often helps
to simplify the solution to certain problems--- it does require caution when it comes to defining
your base case. One part of the coding portion further developed the machine learning algorithm.
The other used recursion and the new topic of numpy arrays to solve a vertical percolation problem.
It's a simple problem in theory but the use of numpy is infinitely useful for many science fields.
